**名称:** GoNeSh (読み: ゴネシュ)

**コンセプト:** "Prompt is the Desktop"

AI時代に特化したエンジニア向け次世代シェル環境。GUIの視認性とCLIの高速操作性を融合し、AI（Gemini/OpenAI/ローカルLLM）とリモートサーバー管理（SSH）を単一の「Go製バイナリ」で完結させる。

---

## 1-1. ターゲット

個人開発者（将来的にチーム利用機能を拡張予定）

---

## 1-2. コアバリュー

| バリュー | 説明 |
|---------|------|
| **Lightweight** | Go言語特有の並行処理とシングルバイナリによる圧倒的な軽快さ |
| **Portal Mode** | ローカルとリモートの境界をなくすシームレスなSSH体験 |
| **Unified View** | 画面上下の視線移動を減らし、全てのステータスを下部に集約 |
| **Multi-Session** | タブ/ペイン分割による複数セッションの並行管理 |

---

## 1-3. 機能一覧

### 1-3-1. シェル基本機能

| 機能 | 説明 |
|------|------|
| **TUIベースのシェル** | Bubbletea製の高速ターミナルUI |
| **タブ管理** | 複数セッションをタブで切り替え |
| **ペイン分割** | 画面を分割して並行作業 |
| **コマンド履歴** | 永続化された履歴と`Ctrl+R`検索 |
| **ステータスバー** | CPU/GPU/メモリをリアルタイム表示 |

### 1-3-2. SSH・リモート機能 (Portal Mode)

| 機能 | 説明 |
|------|------|
| **SSH接続** | 秘密鍵認証によるSSH接続 |
| **Tailscale連携** | MagicDNSによるホスト名解決 |
| **Agentless監視** | リモートリソースをエージェントなしで取得 |
| **Quick Transfer** | `push`/`pull`コマンドで簡単ファイル転送 |
| **ビジュアルファイルブラウザ** | 2ペインでローカル↔リモート対照表示 |
| **Smart Context Switching** | Prod/Dev環境の色分け警告 |

### 1-3-3. AI連携機能

| 機能 | 説明 |
|------|------|
| **マルチAIプロバイダー** | Gemini / OpenAI / ローカルLLM対応 |
| **AIパネル** | 右側スプリット表示でAI応答確認 |
| **プリセットシステム** | 事前定義した指示をスラッシュコマンドで呼び出し |
| **コンテキスト転送** | 選択テキストをAIに即座に送信 |
| **コード生成・レビュー** | `/review`等のプリセットでコード品質向上 |
| **Claude Code連携** | ワンキーでClaude Codeに指示送信 |
| **AIツール統合** | Cursor、Cline等の外部AIツールとの連携 |
| **AIオーケストレーション** | 内蔵AIが状況判断し最適なツールを自動起動 |
| **Git Auto Commit** | AIが変更を解析しコミットメッセージを自動生成 |

### 1-3-4. API Client機能 (Postman風)

| 機能 | 説明 |
|------|------|
| **Swagger/OpenAPI読み込み** | 仕様ファイルからエンドポイント一覧生成 |
| **リクエスト送信** | GET/POST/PUT/PATCH/DELETE対応 |
| **環境変数** | local/staging/production等の環境切り替え |
| **コレクション保存** | よく使うリクエストを保存・再利用 |
| **リクエスト履歴** | 直近のリクエストを自動保存 |
| **Mock Server** | Swagger仕様からモックサーバー自動起動 |
| **Proxy Mode** | 既存APIへのリクエストをログ記録 |

### 1-3-5. UI/UX

| 機能 | 説明 |
|------|------|
| **Hyprland風デザイン** | モダンでミニマルな美学 |
| **フローティングメニュー** | 角丸のオーバーレイUI |
| **`g+キー`ショートカット** | 独自の同時押しキーバインド |
| **Atomic Design** | 再利用可能なコンポーネント設計 |
