## 設定ファイルが見つかりません

### 概要

GoNeShの設定ファイルが指定されたパスに存在しません。

### 原因

- `~/.gonesh/` ディレクトリが存在しない
- 必要な設定ファイル（presets.yaml、connections.yaml等）がない
- ファイルパスの指定が間違っている

### 解決方法

1. 設定ディレクトリを作成する:
```bash
mkdir -p ~/.gonesh
```

2. 必要な設定ファイルを作成する:
```bash
touch ~/.gonesh/presets.yaml
touch ~/.gonesh/connections.yaml
```

3. または、GoNeShを初回起動時に `--init` オプションで初期設定を作成:
```bash
gonesh --init
```
