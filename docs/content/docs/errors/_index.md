GoNeShのエラーコード一覧です。エラーが発生した際は、コードを確認して対処してください。

## エラーコード体系

| カテゴリ | コード範囲 | 説明 |
|---------|-----------|------|
| 設定エラー | E1xxx | 設定ファイル関連 |
| UIエラー | E2xxx | ターミナル・描画関連 |
| SSH/Portalエラー | E3xxx | SSH接続・転送関連 |
| AIエラー | E4xxx | AI API関連 |
| API Clientエラー | E5xxx | HTTPリクエスト関連 |
| Gitエラー | E6xxx | Git操作関連 |
| システムエラー | E9xxx | その他システム関連 |

## 設定エラー (E1xxx)

| コード | 説明 |
|--------|------|
| [E1001]({{< relref "E1001" >}}) | 設定ファイルが見つかりません |

## SSH/Portalエラー (E3xxx)

| コード | 説明 |
|--------|------|

## AIエラー (E4xxx)

| コード | 説明 |
